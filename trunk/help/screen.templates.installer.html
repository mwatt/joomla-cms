Template Installer


-- Draft 4.5.3

There are two ways to install a Template.

Upload Package File

Providing you have GZip support with PHP, you will be able to upload a zipped package file that will be automatically installed to your server.

Package File: Browse to the location of the element on your computer.

Overwrite existing files: This option will replace the existing files with the new ones.

Back-up existing files: This option will make a backup of existing files. This back-up will be written into the administrator/backups folder.

Backup suffix: Type here the suffix you like for your back-up.

Then click the 'Upload' button.

Install From Directory

If you don't have GZip support then you may upload an unpacked archive to your host's server.

Install Directory: Type the location of the folder containing these files (it must be the absolute location).

Then click the 'Install' button.

JS Tree

This navigation tree provides shortcuts to access various aspects of the Administration related to Templates.
Site links to the Site Template Manager. This is where you may Publish, Delete, Edit, edit the XML file or make a package for a site Template.
Admin links to the AdminTemplate Manager. This is where you may Publish, Delete, Edit, edit the XML file or make a package for aa admin Template.
Edit Positions lets edit/create modules positions for the template.
Preview Positions lets preview the home page with details of the positions in use.
Site Preview in New Window lets preview the site in a new window.
Site Preview Inline lets preview the site in aFrame in the admin back-end.
List packages links to a list of all Templates packages available on your site.
