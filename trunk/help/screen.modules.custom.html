Module: Custom

4.5.3

The Custom Module is created when clicking the 'New' icon in the Module Manager.
You may use this module to display Content (this content is not related to Sections, Categories or Static Content Items).

Type: custom

Details

Title: Type the name of the title of the module that you wish to display on your Web page.

Show title: Choose if you wish the Title to be published or not on your Web page.

Position: Choose the position of your module.  NOTE:  The position must be available in your template.

Module order: Choose the order that you wish this module to appear among modules assigned to the same position on the same Web page.

Access Level: Choose the access level for your visitors to view this module. Public, Registered or Special.

Published: Choose if this module's contents will show on the Frontend of your Web site.
NOTE: The Position of this module must be available in your template.

ID: This is the ID number for this module. This is auto-allocated by the Joomla! system when the module is saved.

Pages / Items

Choose which pages you wish to publish this module.

Parameters

Module Class Suffix: Type, if needed, the suffix to be applied to the CSS class of the module (table.moduletable) to allow individual module styling.

Custom Output: Enter any text or code that you would like to show up in this module postion on your Web page.  (This content cannot be not be accessed through a Menu Item).
