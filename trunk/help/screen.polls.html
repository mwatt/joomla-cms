Poll Manager

-- Draft 4.5.3

The Poll Manager shows a list of the Polls you may display on your site.
The Polls are displayed using the Polls Module (mod_poll).
Help: Poll Module

Toolbar Icons

Select Polls by clicking the checkbox at the start of each row.

Publish and Unpublish: Click the 'Publish' or 'Unpublish' icon to publish or unpublish any number of selected Polls. You may check multiple checkboxes.

Delete: Click the 'Delete' icon to delete any of the Polls.
 
Edit: Click the 'Edit' icon to edit the Poll.

New: Click the 'New' icon to create a new Poll.

Filters

From left to right:
Search a Poll by name.
State: displays depending on Publish or Unpublished state.

Column Descriptions

Clicking on any title will order the items according to the title clicked.

Check Box/Padlock: Click this to select the item. A padlock icon will replace the Check Box if the item is checked-out. Clicking on it will check-in the item on an individual basis. If this item has been checked out by another manager/administrator, please contact the administrator of your site for him/her to Check-in the item. When checked, clicking the Toolbar Icons will perform the various actions available.

Poll Title: This is the name of the poll. If you click on this name, you may edit the related parameters.
 
Published: This shows whether the poll is published. Click the icon to toggle the state of the poll.
 
Options: This is the number of options in the poll.

Lag: Time in seconds between votes of the same user for the same poll.

ID: There is the ID for each Poll as defined in the database.

At the bottom of the list you may see a pagination feature, which will allow you to navigate to the next set of items in the list. You can also set the number of items displayed on a page by selecting the number in the drop down box at the bottom of the screen.
