Language Installer

There are two ways to install a Language.

Upload Package File

Providing you have GZip support with PHP, you will be able to upload a zipped package file that will be automatically installed to your server.

Package File: Browse to the location of the element on your computer.

Overwrite existing files: This option will replace the existing files with the new ones.

Back-up existing files: This option will make a backup of existing files. This back-up will be written into the administrator/backups folder.

Backup suffix: Type here the suffix you like for your back-up.

Then click the 'Upload' button.

Install From Directory

If you don't have GZip support then you may upload an unpacked archive to your host's server.

Install Directory: Type the location of the folder containing these files (it must be the absolute location).

Then click the 'Install' button.

JS Tree

This navigation tree provides shortcuts to access various aspects of the Administration related to Languages.
Site links to a table displaying Site files for a specific language installed on the site.
Administrator links to a table displaying Administrator files for a specific language installed on the site.
Web installer links to a table displaying Install files for a specific language installed on the site.
Install links to the Language Installer.
List Packages display a table list of all packages available for your site (installed or not).
Trawl lets your search the Language strings present in the Joomla! Translation Functions for no corresponding entries in the Language INI files.
