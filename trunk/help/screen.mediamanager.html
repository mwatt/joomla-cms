Media Manager

-- Draft 4.5.3

The Media Manager enables you to organise your media files within Joomla! for use in various articles on your Web site.

Toolbar Icons

Folder: Click this icon to display a field where you may enter the name for a new folder to be created in the directory displayed on the page. A "close" button lets you close the frame. A message confirming the success of the operation will display and the folder will appear in the directory chosen.

Upload: Click this icon to display a field where you may enter the files location or click the 'Browse' button for a local file to upload to the directory displayed on the page. A "close" button lets you close the frame. A message confirming the success of the operation will display and the file will appear in the directory chosen.

Files: ./"name of directory"

The title changes to the name of the directory displayed.

Clicking on a folder will display this folder's content.

Clicking a thumbnail will display the image at real size in a pop-up window.

Code Generated: Clicking the pencil icon under it in the Media Manager window will enter the hyperlink code in the 'Code' text field. This can copied for pasting, for example, into a News item or other item appearing on the Frontend of your Web site.

Delete: To delete a file or a directory, click the 'Trash Can' icon under the image of the file or directory in the Media Manager window. A directory must be empty before it can be deleted.

JS Tree

This navigation tree provides shortcuts to access the existing Media directories on your site.
