User Manager

-- Draft 4.5.3

The User Manager displays a list of the Users registered on your site.  You can add or edit Users here.

Toolbar

Select users by clicking the checkbox at the start of each row.

Logout: Click the 'Logout' icon to log a user out.

Delete: Click the 'Delete' icon to delete the user from the database. You can delete more than one user at a time.

Edit: Click the 'Edit' icon to edit a User's information.

New: Click the 'New' icon to create a new User.

Help: Users Edit/New

JS Tree

This navigation tree provides shortcuts to display Users in function of their access levels.
It is divided in 2 parts : Frontend and backend.
Help: Users Access levels.

Filters

From left to right:
Search a User by name.
Select Group: lets display users according to their access level.
Select Log Status: lets display users according to their logged status.

Column descriptions

Clicking on any title will order the users according to the title clicked.

Check Box/Padlock: Click this to select the user. A padlock icon will replace the Check Box if the user is checked-out. Clicking on it will check-in the user on an individual basis. If this user has been checked out by another manager/administrator, please contact the administrator of your site for him/her to Check-in the user. When checked, clicking the Toolbar Icons will perform the various actions available.

Name: This is the real or display name of tye  user. Click the name to edit the User's information. A padlock icon next to it shows that it is checked-out. To check-in, click the name to edit.

Username: This is the login name of the user.

Logged In: This shows whether the user is logged in or not.

Enabled: This shows whether the user is allowed to log in to the site. Click the icon to toggle the state of the user.
 
ID: This is the ID of the user in the database.
 
Group: This is the group that the user is associated with.
 
Last Visit: This is the date and time that the user last visited a page on the site while logged in.

At the bottom of the list you may see a pagination feature, which will allow you to navigate to the next set of banners in the list. You can also set the number of banners displayed on a page by selecting the number in the drop down box at the bottom of the screen.
