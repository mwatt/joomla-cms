Mambot: Load Module Positions

Load Module Positions mambot.


Loads module positions within Content.
Usage: {mosloadposition user1} where 'user1' is one of the positions defined for your site through Site->Templates->Modules Positions.

This means you may load any module assigned to this position at this specific place into your Content without defining the position into the Template index.php file.
Choose a style of HTML code to insert the module via the parameters.

Toolbar Icons

Save: Click this icon to save the parameters entered and return to the Mambot Manager.

Apply: Click this icon to save the parameters entered AND keep the same window open for editing.

Close: Cancels the action. Any modification is lost.

Mambot Details

Name: Type here the name of the Mambot as it will appear on the Site Mambots listing. This has no impact on your site.

Folder: The Category into which this mambot is installed.

Mambot file: The name of the Mambot .php file as it appears in the folder. Do not change.

Order: You may change here the order of the Mambot as listed in the Folder when filtering by category.

Access Level: Choose who will be able to view this content. Public, Registered or Special.

Published: Check the box to let the Mambot show on your site or provide the function it is designed to.

Description:  Loads module positions within Content. Syntax: {mosloadposition user1}

Parameters

Style:  Choose the kind of code to wrap the module.
* Wrapped by Table - Column
* Wrapped by Table - Horizontal
* Wrapped by Divs
* No wrapping, raw output
