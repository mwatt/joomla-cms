Template Manager

-- Draft 4.5.3

The Template Manager lets you assign and edit Templates, as well as creating templates packages
 

Toolbar Icons

Select a template by checking the radio button next to its name.

Packages: Creates a package with all items included for easy distribution on the Net.

Assign: Opens a page where you can assign this template to specific Menu Items/pages. This will override de default template for the pages chosen. If you choose 'None', it will act the same way as the 'Default' parameter.
This function is not available for Administrator templates.

Default: Assigns this template to all Menu Items/pages per default.

NOTE: Both the assigned and the default template will get overridden by the Template Chooser module in the Frontend, if used. Any template displayed on this list will show in the Template Chooser module.
 
Edit HTML: Lets edit the templates/the-template-chosen/index.php file. This is the file loaded in the FrontEnd when a user opens your site.

Edit CSS: Lets edit the templates/the-template-chosen/css/template_css.css file which contains the CSS classes definitions for this specific template
 
Delete: Lets you delete a template. You will be asked to confirm the delete operation. Select 'Yes' to delete or 'No' to cancel. The template will be definitively deleted from the database.

Edit XML: Lets edit the xml file for a template. This includes metadata and the listing of files necessary to make a package.
 
The Administrator Templates can be choosen by clicking the "Default" icon. Administrator Templates are not assignable to any pages.

JS Tree

This navigation tree provides shortcuts to access various aspects of the Administration related to Content Items.
Site links to a list of the Site templates available on your site.
Admin links to a list of the Administrator templates available on your site.
Edit Positions lets edit/create modules positions for the template.
Preview Positions lets preview the home page with details of the positions in use.
Site Preview in New Window lets you display your home page in a new window.
Site Preview Inline lets you preview your home page in a frame in the admin interface.
Install links to the Template Installer.
List Packages links to a page listing all Templates packages available on your site.

Column Descriptions

Name: This is the name of the template. If you mouseover this name —and the 'Preview template' on top right of the list is checked—, a preview of the template is displayed. If you click on the name, this template will be set as default.
 
Default: This is the template that is currently published as the site template (only one can be published at a time).
 
Assigned: This tells us whether the template is dynamically assigned to one of the pages on the site.
 
Version: This is the template version.
 
Date: This is the creation date of the template.

Author: The name of the template's author.
 
Web site: A URL provided by the author.

E-mail: an e-mail provided by the author.

At the bottom of the list you may see a pagination feature, which will allow you to navigate to the next set of modules in the list. You can also set the number of modules displayed on a page by selecting the number in the drop down box at the bottom of the screen.
