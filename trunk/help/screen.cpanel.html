Control Panel

-- Draft 4.5.3

The Control Panel is the first page you see when you log into the Administrators area. You can return to it by clicking the "Home" menu item in the top menu bar. 

Quick Icons

From the Control Panel are 10 shortcut icons to easy access the frequently used parts of the Administrator site.

Add New Content  - bypasses the Content Items Manager by displaying directly the New Content Item screen.

Content Items Manager  - lets access to a table list of all Content Items on your site. There you may create/edit content in any category already defined.

Static Content Manager  - lets access to a table list of all Static Content Items on your site. There you may create/edit any of these contents. They do not belong to any category.

Frontpage Manager -  lets access to a table list of all Content Items assigned to the FrontPage Component.  You will be able to edit which Content items appear through this Component.

Section Manager - lets access a table list of all Sections on your site.  You will be able to create/edit your Sections.

Category Manager - lets access a table list of all Categories on your site.  You will be able to create/edit your Categories.

Media Manager - lets access to a screen to manage images and documents to use on your site. There you may upload, delete, organise, get links concerning various medias.

Menu Manager - lets access to a table list of all menus defined on your site.  There you may  create/edit any of your menus. 

User Manager - lets access to a table list of all users registered on your site. There you may edit/create users depending on your access level.

Global Configuration - lets access to a tab view of a number of parameters concerning your site.  There you may edit any of your Web site configurations.

Information Tabs

On the right side of the screen are a number of tabs. These are Administrator Modules. They display the following information, depending on the Modules published:

Logged - This displays a list of users currently logged on (both in the Admin and the Frontend sections). Click on a users name to go to the User Edit page for that user.

Components - Click this shortcut tab to access any of the components and component links that you wish to edit.

Popular - Click this tab to access the screen which lists in order, the most popular Content items on your site.  This includes the date the Content item was created and the number of  hits. Click the Content Item name to edit that item.

Latest Items - Click this tab to access the screen which lists the most recently added Content items.  This includes the creation date and the author. Click the Content item name to edit that item.

Menu Stats - Click this tab to access the screen which lists all menus and the number of item links in each menu.  The number of links is the total created, not the total published. Click the name of a menu to access the screen which lists the items for that menu.

By editing the Modules::admin, you may publish/add other modules to display with their own tabs.
