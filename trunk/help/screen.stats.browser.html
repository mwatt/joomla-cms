Statistics: Browser, OS, Domain

draft 4.5.3

This screen displays a list of Statistics concerning the site's users since you last reset it.
Staistics available concern the Browser type, the OS used as well as the Domain the user comes from.


The Global Configuration->Statistics should be set to "yes" to get this information.

Toolbar Icons

Delete: resets the count to zero for all statistics.

Statistics Links

This navigation tree provides shortcuts to access various aspects of the Administration related to Statistics.

Tabs

Click the tabs to get the statistics to show for the tab title.

At the bottom of the list you may see a pagination feature, which will allow you to navigate to the next set of items in the list. You can also set the number of items displayed on a page by selecting the number in the drop down box at the bottom of the screen.
