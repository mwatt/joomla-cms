Elements: List packages

draft 4.5.3

This page lists all packages available on your site.
Packages are ready-to-install compressed files to distribute on the Net.

The elements concerned may be Components, Mambots, Modules, Languages or Templates.

Toolbar Icons

Select packages by clicking the checkbox at the start of each row. You may batch-delete by selecting all packages.

Delete: lets you delete the package(s).

Close: cancels the action and return to the Manager for the concerned element.

JS Tree

This navigation tree provides shortcuts to access various aspects of the Administration related to the elements concerned.
