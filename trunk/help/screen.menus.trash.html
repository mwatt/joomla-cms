Trashed Menu Items

-- Draft 453

This screen shows a list of the Trashed Menu Items on your site.

Toolbar Icons

Select items by clicking the checkbox at the start of each row.

Restore: click this icon to restore the item in its original menu.

Delete: click this icon to delete an item PERMANENTLY from the database. You will be asked to confirm the delete operation. Select 'Yes' to delete the content item or 'No' to cancel.

Navigator

This navigation tree provides shortcuts to access various aspects of the Administration related to Menus.
Menu Manager links to the Menu Manager
Menus links to each menu defined on the site.

Column Descriptions

Clicking on any title will order the items according to the title clicked.

Title: This is the title of the Menu item.

Menu: This is the Menu the item belongs to.

Type: This is the type of Menu Items concerned
 
ID: There is the ID for each menu itemt as defined in the database.

At the bottom of the list you may see a pagination feature, which will allow you to navigate to the next set of items in the list. You can also set the number of items displayed on a page by selecting the number in the drop down box at the bottom of the screen.
