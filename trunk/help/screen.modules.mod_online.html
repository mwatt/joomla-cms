Module: Online Users

4.5.3

The Online Users Module displays the # of online users in the header on the Administrator interface.

Type: mod_online

Details

Title: Title of the module.

Position: Sets which of the positions available the module will display in. Header is default.

Module order: The order the module will appear among modules assigned to the same page, in the same position. Second in the list is default.

Published: Sets if the module contents will show on the Backend or not. It is advised to let it published.

ID: This is the ID number for the module item. This is auto-allocated by the Joomla! system when the module is saved
