User Manager - Add or Edit Users

-- Draft 4.5.3

This screen is used to either add a new user or edit an existing one.

Toolbar Icons

Save: Lets save the creation/modification and return to User Manager.

Apply: Saves the creation/modification and keeps the screen open for editing.

Cancel: Cancels the action and lets the User checked-in.

User Details

Name: Type here the name of the user as it will be listed in the User Manager.

Username: Type here The Username as it has to be entered for login.

Email: Type here the user's e-mail adress.

New Password: Type here the user's password.

Verify Password: Type here the user's password.

Group: Choose here a type of user.
Help Screen: User Access Control Levels

Block User: Prevents or lets user login.

The following info is displayed when a new user is saved. or when an existing user is edited.

Receive Submission Emails: Enable or not an e-mail be sent to the administrator when the user has submitted some content. Depends on the Access level.

Register Date: Displays the date the user has been registered.

Last Visit Date: Displays the date on which this user has last logged in.

Parameters

Administrators may select here the language they want to be used in the frontend and backend, depending on the languages installed on the site.
From Authors up, they may choose also the Editor they want to be used among those installed.

These parameters will overcome the Global Configuration default Site parameters for Language and Editor.

If the "Frontend User Params" is set to 'Yes" in the Global Configuration->User tab, a Registered user will be able to edit these parameters from the Front-end when editing her/his Details.

Contact Information

These informations are only present when editing a user and only if the contact has been created in the Contacts Component and linked to the user (Components -> Contact -> Manage Contacts). If this has been done the information will be displayed and a link at the bottom of the box provides a direct access to the Edit Contact page for that contact.
