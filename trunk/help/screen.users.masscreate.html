User manager: Mass Create Users

draft 4.5.3

This page lets create as many as 10 users at once, sending them by mail their random generated password when saved.

Toolbar Icons

Save: saves the users created in the database and send each of them a mail with her/his username and password.

Cancel: cancels the action and return to the User Manager.

Group

Choose here the type of users you want to create.
Help: User Types

User Details

Name: Enter in each Name field the name of the user. It will also become the "username" when logging.

E-mail: Enter in each field the user's e-mail. The user will receive a mail with a random generated password and her/his username when saving.
