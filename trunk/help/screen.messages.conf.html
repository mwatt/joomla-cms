Messages Configuration

-- Draft 4.5.3

This page lets configure the way System Messages are sent to the Administrator.

Toolbar Icons

Save: Saves the configuration and returns to Control Panel.

Apply: Saves the configuration and keep the page displayed to edit.

Close: Cancels the action. Any modification is lost.

Settings

There are two configuration options available.

Lock Inbox: This option allows to prevent Administrators or System processes from sending you messages. If you are the only administrator or work with a small group of admins you may choose not to use this feature at all. It is a good idea to leave the radio button set to 'Yes' while you are adding new components as it will help you be aware of system processes.
 
Mail me on new Message:  this option will result in an e-mail being sent to the Administrator each time a new message is recieved.
