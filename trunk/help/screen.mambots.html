Mambot Manager

---- Draft 4.5.3

The Mambot Manager displays a list of the Mambots installed on your site.

Mambots are essentially code snippets which when called via various parameters either activate a program, script or perform a specific function. They can be as simple as a function which replaces typed text with code (such as typing to display an image in a content item) to full fledged third party libraries such as WYSIWYG editors or other programs.

[Detailed functions for default Mambots may be found here:
Help?Mambots: New/Edit]

Mambots are divided in 4 types/categories you may display by filtering.

Toolbar Icons

Select a Mambot by clicking on the checkbox next to its name.

Publish/ Unpublish: Click the icon to Publish or Unpublish the Mambot.

Packages: Creates a package with all items included for easy distribution on the Net.

Delete: Click the "Delete" icon. This will uninstall the Mambot.
WARNING: This cannot be undone. Deleted Mambots cannot be restored as a "Menu Item" or "Content Item" can. If you have deleted a Mambot and you wish to use that Mambot again, you must re-Install the Mambot.
NOTE: Mambots essential to core operations cannot be deleted.

Edit XML: Lets edit the xml file for a Mambot. This includes metadata, the listing of files necessary to make a package and various parameters.

Edit: Click the "Edit" icon to change the settings as desired.

New:  Click the "New" icon to create a new Mambot. Then fill in the fields listed.

JS Tree

This navigation tree provides shortcuts to access various aspects of the Administration related to Site Mambots.
Content, editors, editors-xtd, search, system and user filter the Mambots display according to these groups.
Install links to the Mambot Installer.
List Packages lets display a list of all Mambots Packages available on your site (installed or not).

Filters

From left to right:
Search a Mambot by name.
State: displays depending on Publish or Unpublished state.
Access: displays depending on the access level.

Column Descriptions

Clicking on any title will order the items according to the title clicked.

Check Box/Padlock: Click this to select the item. A padlock icon will replace the Check Box if the item is checked-out. Clicking on it will check-in the item on an individual basis. If this item has been checked out by another manager/administrator, please contact the administrator of your site for him/her to Check-in the item. When checked, clicking the Toolbar Icons will perform the various actions available.

Published : This shows whether the Mambot is published or unpublished. By clicking on the icon, you may switch its state.

Reorder :  Click the arrow icons to move the item up or down in the list. This has no effect on the frontend of your site.

Order : Shows the order of Mambots in relation to other Mambots of the same type. You may change it by editing 2 # conjointly and clicking on the ‘Save Order’ icon next to the Column Heading ‘Order’. This has no effect on the frontend of your site.

Access: This shows who can see or use the Mambots on the Frontend of a Web page. Clicking the link will toggle through Public/Registered/Special.

Group: Displays the type of mambot, types include Content, editors, editors-xtd, search, system and user.

File: Displays the mambots filename.

ID: This is the Mambot's ID in the database.

At the bottom of the list you may see a pagination feature, which will allow you to navigate to the next set of items in the list. You can also set the number of items displayed on a page by selecting the number in the drop down box at the bottom of the screen.
