Trawl

draft 4.5.3

Trawl is a function used during the development of either the core Joomla! code, or by custom developers to aid the initial preparation of ini help files. Trawl finds new strings that have been translated in your code. Cut and paste them into the appropriate language file.

Toolbar Icons

Trawl: Click the 'Trawl' icon to Initiate the file search (trawling).

Options

The following options are available:
Search: Select the radio button to search for the Site, the Administrator or the Web Installer.

Results Screen

The results screen is broken into two 'tabbed' areas.

New Tab

Grouped by file, this tab shows you strings that are passed to any of the Joomla! translation functions (either the $_LANG->_ method call or patTempalte's <mos:Translate> tag) but do not have a corresponding entry in the ini language file.

Other Tab

This shows entries in the ini languages files that are possibly not used (you need to confirm this by hand though).
