Mass Mail

-- Draft Version

This page lets send a mail to one or more groups of Users.

Toolbar Icons

Send Mail: Once the mail is ready, click this icon to send it. The screen remains open to send another mail.

Close: Closes the screen and return to Control Panel.

Details

Group: Make a selection from the list. Select "All User Groups" to mail the message to all users. Only one selection can be made from this list.
 
Mail to Child Groups: Check this box to automatically include all of the child groups of the selected group. For example, if you select the "Editor" group and check this box, all users in the Publisher group will also be included in the mass mail.

Send in HTML mode: Choose this setting if you paste an HTML formatted content in the Message field.
 
Subject: Type a subject for the message.
 
Message: Type the body of the message or paste some HTML formatted content.
 
When you are finished, click the Send Mail icon to mail the message or Close to abort the mail.
 
NOTE: Only a user in the Super Administrator group is permitted to use this feature.
