Module: Menu Stats

4.5.3

The Menu Stats Module  shows in the Control Panel tabs on the Administrator interface.
It displays the Menus that are defined on your site as well as the number of Menu Items for each.
Clicking on the Menu name lets edit the Menu.

Type: mod_menustats (admin)

Details

Title: Title of the module.

Position: Sets which of the positions available the module will display in. Cpanel is default.

Module order: The order the module will appear among modules assigned to the same page, in the same position. Sixth in the list is default.

Published: Sets if the module contents will show on the Backend or not.

ID: This is the ID number for the module item. This is auto-allocated by the Joomla! system when the module is saved
