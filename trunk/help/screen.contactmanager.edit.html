Contact Manager : New / Edit

--- Draft 4.5.3

This page lets you create or edit contacts on your site.

Toolbar Icons

Save: Saves your contact Details and returns to Contact Manager.

Apply: Saves your contact Details and keep the editing screen displayed for editing.

Cancel : Exits the screen without saving. This takes back to Contact Manager. The contact remains checked-in.

Contact Details

Any field left blank is not displayed.
Any field filled out will be included into the downloadable vCard for this Contact.

Category: Drop-down select of all published Contact categories

Linked to User: Drop-down select of all registered users of the website. Can be set to " - No User - " for independent setup of a site contact.

Name: Type here the name of the contact as it will display on the site. Can be either a person's name, or a departement, a product, or whatever you wish.

Contact's Position: Best used with a Name, as the title is displayed if this field is used. Example would be "CEO".

E-mail: E-mail adress for this Contact.

The following 7 fields can be used to provide geographical access information if desired.
  • Street Address
  • Town/Suburb 
  • State/County
  • Country 
  • Postal Code/ZIP
  • Telephone  
  • Fax
Miscellaneous Info:  Type here any other information you wish to provide about this Contact.

Publishing Info

Published: Select Yes or No to show this Contact to your visitors.

Ordering: Contact needs to be saved/applied first to display a drop-down select here to order this Contact in the list of published Contacts for the site. Select the Contact after which you wish to display the current Contact.

Access: Select here who can view this Contact on the frontend if published. Public, Registered or Special

Images Info

You may select here an image (from folder images/stories) to display next to this Contact, or select "- Select Image -" to not display any image.

If an image is chosen, it will show in the "Preview" block.

Parameters

Here are various settings for what is displayed when the user clicks to view a Contact item.

Page Class Suffix: Allows you to set customized styles for this page. The suffix you type here is automatically added to the CSS classes for this page.

Print Icon: Hides or shows the icon and link to the printer-friendly format of items. The pull down lets select 'Use Global' to use the parameter as defined globally through the Site->Global Configuration, 'Hide' or 'Show'.

Back Button: Controls whether to show or hide a "Back" text link that would appear on the bottom of the page and return the user to the previously viewed page. The pull down lets select 'Use Global' to use the parameter as defined globally through the Site->Global Configuration, 'Hide' or 'Show'.

The following 12 fields defined above may be hidden or shown by checking the radio buttons:
  • Name
  • Position
  • E-mail
  • Street Address
  • Town/Suburb 
  • State/County
  • Country 
  • Postal Code/ZIP
  • Telephone
  • Fax
  • Misc. Info
  • Image
--------------------------
E-mail description: Show/Hide the description text below.

Description text: Type here the description text for the Email form. If left blank, it will use the _EMAIL_DESCRIPTION language definition.

E-mail Form: Show/Hide the e-mail to form.

E-mail copy: Show/Hide the checkbox to e-mail a copy to the sender's adress.

---------------------------
Drop-down: Hide or Show the drop-down select list when in full view. Whatever the contact link accessed, this will let the user choose another contact to display.

---------------------------
Vcard: Hide/Show the Vcard download function.

----------------------
Icons/Text: This lets you decide if the fields listed below should be introduced by an icon, the default text for the fields in the Contact Details or nothing.

You may choose, if decided so above, a specific icon to introduce the following fields:
  • Address
  • E-mail
  • Telephone
  • Fax
  • Misc Info
If you choose a custom image, this image should be uploaded to the images/stories directory in the Art Manager.
