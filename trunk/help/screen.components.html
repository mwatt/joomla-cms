Component Manager

-- Draft 4.5.3

This page displays the Components installed on your site.

Components are content specialised elements or applications. Mambo includes some core components and supports the possibility to enhance its features with additional 3rd party components.

Here you may make packages, edit the related XML files or delete components.

Toolbar Icons

Select Components by clicking the radio button at the start of each row.

Package: lets you make a package of a component to distribute on the Net.

Edit XML: lets you edit and save the component XML file.

Delete: deletes the Component. A javascript alert will show: "Are you sure you want to delete selected items?"
Click 'OK'. The page will reload displaying the word 'Success'.
NOTE: The components are deleted from the database. They cannot be recovered from the Trash Manager.

JS Tree

This navigation tree provides shortcuts to access various aspects of the Administration related to Components.
Components links to a table display of all components installed on your site and links to functions available.
Installer links to the Component Installer.
Create links to the Create Component page
List packages links to a list of all Components packages available on your site.

Column Descriptions

Clicking the radio-button will select the Component.

Name: This is the name of the Component. Clicking on the name lets edit the parameters for that component.
 
Option: displays the file name for the component.
 
Version: The version # for the component.
 
Date: This is the creation date of the Component.

Author: The name of the component's author.
 
Web site: A URL provided by the author.

E-mail: an e-mail provided by the author.

At the bottom of the list you may see a pagination feature, which will allow you to navigate to the next set of components in the list. You can also set the number of components displayed on a page by selecting the number in the drop down box at the bottom of the screen.
