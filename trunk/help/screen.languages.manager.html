Language Manager

-- Draft 4.5.3

The Language Manager lets edit, add and package languages.

 As of 4.5.3, the Installation, Back-end and Front-end is fully translatable.

Toolbar Icons

Select files by checking the box at the start of each row.

Package: Select all files by checking the box in the table header and click this icon.  Packaging lets assemble the package file that allows you to share the language files on the Web. Each language needs three packages: Install, Site and Admin.

Delete: Click this icon to delete the file(s).

Edit: Click this icon to edit a file.

Navigator

Site links to a table displaying Site files for a specific language installed on the site.
Administrator links to a table displaying Administrator files for a specific language installed on the site.
Web installer links to a table displaying Install files for a specific language installed on the site.
Install links to the Language Installer.
List Packages display a table list of all packages available for your site (installed or not).
Trawl lets your search the Language strings present in the Joomla! Translation Functions for no corresponding entries in the Language INI files.

Create

Lets create a new language group of files from scratch or as a copy of an existing language.

Enter the name of the new language file (this should be lower case and not contain any spaces).  You may choose to copy from an existing language or just create blank language file to start afresh.  Click the 'Create' button to generate the new language files.

Column Descriptions

Select a file by checling the box next to its name.

File: This is the name of the language file.  The primary language will be called language.ini (for example, english.ini) and this contains many common words and phrases used by many components.  The other language files are for individual components.

Size: This is the size of the file, in bytes.

Last Modified: This is the date and time that the file was last modified on the file system.
