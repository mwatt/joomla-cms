Banner: Client Manager

-- Draft 453

This page shows the list of Clients whose banners may be displayed on your site.

Toolbar Icons

Select clients by clicking the checkbox at the start of each row.

Delete: Click this icon to delete a client. You will be asked to confirm the delete operation. Select "Yes" to delete the client or "No" to cancel.

Edit:  Click this icon to edit the client's details.

New:  Click this icon to add a new client.

JS Tree

This navigation tree provides shortcuts to access various aspects of the Administration related to Banners.
Banners links to the Banner Manager
Clients links to the Clients Manager
Media links to the Art Manager

Column descriptions

Clicking on any title will order the items according to the title clicked.

Check Box/Padlock: Click this to select the client. A padlock icon will replace the Check Box if the client is checked-out. Clicking on it will check-in the client on an individual basis. If this client has been checked out by another manager/administrator, please contact the administrator of your site for him/her to Check-in the client. When checked, clicking the Toolbar Icons will perform the various actions available.

Client Name: This is the name of the client.

ID: The internal database identification for the client.

Contact: Displays who is in charge for this client.

E-mail: displays the e-mail for the client.

At the bottom of the list you may see a pagination feature, which will allow you to navigate to the next set of clients in the list. You can also set the number of clients displayed on a page by selecting the number in the drop down box at the bottom of the screen.
