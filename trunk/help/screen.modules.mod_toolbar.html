Module: Toolbar

4.5.3

The Toolbar Module displays the Toolbar on the Administrator interface.

Type: mod_toolbar

Details

Title: Title of the module.

Position: Sets which of the positions available the module will display in. Toolbar is default. No user choice.

Module order: The order the module will appear among modules assigned to the same page, in the same position. Toolbar is default. No user choice.

Published: Sets if the module contents will show on the Backend or not. It is advised to let it published.

ID: This is the ID number for the module item. This is auto-allocated by the Joomla! system when the module is saved.
