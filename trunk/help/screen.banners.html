This page shows a list of the banners you may display on your site.
 

Toolbar Icons

Select banners by clicking the checkbox at the start of each row.

Upload: Opens a new window which lets you upload a file or an image from your computer to the images/stories folder.

Publish and Unpublish: Click the 'Publish' or 'Unpublish' icon to publish or unpublish any number of selected banners.

Delete: Click this icon to delete the banner.

Edit: Click this icon to edit the banner's details.

New: Click this icon to add a new banner.

JS Tree

This navigation tree provides shortcuts to access various aspects of the Administration related to Banners.
Clients links to the Clients Manager
Media links to the Art Manager

Filters

From left to right:
Search a Banner by name.
State: lets display banners according to their published state.

Column descriptions

Clicking on any title will order the items according to the title clicked.

Check Box/Padlock: Click this to select the banner. A padlock icon will replace the Check Box if the banner is checked-out. Clicking on it will check-in the banner on an individual basis. If this banner has been checked out by another manager/administrator, please contact the administrator of your site for him/her to Check-in the banner. When checked, clicking the Toolbar Icons will perform the various actions available.

Banner Name: Click this name to edit the related banner parameters.

Published: This displays whether the banner is currently Published or not for Frontend display. Click the column icon to toggle state. Click the triangle to display all banners by state.

ID: The ID of the banner in the database

Date:  I have no idea what this related to (?).

Impressions Made: This is the number of times the banner has been shown on your site.
 
Impressions Left: This is the number of impressions left to display if a limit has been set while creating or editing a banner.
 
Clicks: This is the number of times that particular banner has been clicked on by a user of the site.
 
% Clicks: This is the number of clicks as a percentage ratio to the number of impressions that have been made. 1% would mean for example, that 1 in every 100 people have clicked on the banner.
 
At the bottom of the list you may see a pagination feature, which will allow you to navigate to the next set of banners in the list. You can also set the number of banners displayed on a page by selecting the number in the drop down box at the bottom of the screen.
