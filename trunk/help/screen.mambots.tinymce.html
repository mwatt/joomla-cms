Mambot: TinyMCE WYSIWYG Editor

draft 4.5.3

TinyMCE WYSIWYG editor is the default editor installed with Mambo.
As of 08/07/05 the version provided is 1.45.

Toolbar Icons

Save: lets save the content and return to the specific Manager.

Apply: lets save the content and keep the editing screen open to edit.

Close: cancels the operation and let the item checked-in if opened for editing.

Mambot Details

Name: Type here the name of the Mambot as it will appear on the Site Mambots listing. This has no impact on your site.

Folder: The Category into which this mambot is installed.

Mambot file: The name of the Mambot .php file as it appears in the folder. Do not change.

Order: You may change here the order of the Mambot as listed in the Folder when filtering by category.

Access Level: Choose who will be able to view this content. Public, Registered or Special.

Published: Check the box to let the Mambot show on your site or provide the function it is designed to.

Parameters

Functionality: choose here between Advanced, Basic and Simple. This may add or take off some editing features.
--------------
Text direction: choose here between Left to right (Western languages) and Right to Left (Arabic, Farsi, Hebrew).

Prohibited elements: type here the kind of code to prohibit from the text entered (example: script, Applets, etc.). Elements shall be separated by comas.

Template CSS classes: displays or not a drop-down withe CSS defined for your site in template_css.css or a specific css file.

Custom CSS classes: if yes is chosen in the former choice, you may enter here the full path to a specific CSS file to present the Classes in a drop-down.

Newlines: Choose here between p or br elements.
--------------------
Toolbar: choose here to display the toolbar on top of the editing field or on bottom of it.
-------------------
Smilies: hides or shows the Smilies buttons. Only works in advanced mode.

Table: shows or hides the Table control buttons. Only works in advanced mode.

Flash: shows or hides the Flash button. Only works in advanced mode.

Horizontal rule: shows or hides the horizontal rule button.
------------------
Fullscreen: shows or hides the fullscreen.

Context menu: Shows or hides a small Context menu within the editor. Only works in advanced mode. As of 08/07/05, the only way to change the language for these menus is to edit the mambots/editors/tinymce.php file around line 147. Change there: language : "en", to the language of your choice present in the mambots/editors/tinymce/jscripts/tiny_mce/langs/ folder.

HTML Height: Height of HTML pop-up window. Only works in advanced mode.

Width: Width of HTML pop-up window. Only works in advanced mode.

Preview: Shows or hides the Preview button. Only works in advanced mode.

Preview Height: Height of preview mode pop-up window. Only works in advanced mode.

Preview Width: Width of preview mode pop-up window. Only works in advanced mode.

Search and replace: Hides or shoes the Search and Replace button. Only works in advanced mode.

Insert Date: Shows or hides the Insert Date button. Works only in advanced mode.

Date format: Format of inserted date. Works only in advanced mode.

Insert Time: Shows or hides the Insert Time button. Works only in advanced mode.

Time format: Format of inserted Time. Works only in advanced mode.
