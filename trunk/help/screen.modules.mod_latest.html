Module: Latest Items

4.5.3

The Latest Items Module  shows in the Control Panel tabs on the Administrator interface.
It displays a list of the most recently published Items that are still current (some may have expired even though they are the most recent). Items that are displayed through the Front Page Component are not included in the list.

Type: mod_latest

Details

Title: Title of the module.

Position: Sets which of the positions available the module will display in. Cpanel is default.

Module order: The order the module will appear among modules assigned to the same page, in the same position. Fifth in the list is default.

Published: Sets if the module contents will show on the Backend or not.

ID: This is the ID number for the module item. This is auto-allocated by the Joomla! system when the module is saved
