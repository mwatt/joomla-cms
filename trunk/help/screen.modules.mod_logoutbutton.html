Module: Logout Button

4.5.3

The Logout Button Module displays in the header on the Administrator interface the name of the user logged in the back-end, as well as a Logout button.

Type: mod_logoutbutton

Details

Title: Title of the module.

Position: Sets which of the positions available the module will display in. Header is default.

Module order: The order the module will appear among modules assigned to the same page, in the same position. Third in the list is default.

Published: Sets if the module contents will show on the Backend or not.

ID: This is the ID number for the module item. This is auto-allocated by the Joomla! system when the module is saved
