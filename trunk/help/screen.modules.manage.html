Module Uninstaller and Package making

Toolbar Icons

Select Modules by clicking the radio button at the start of each row.

Edit XML: lets you edit and save the Module XML file.

Package: lets you make a package of a module to distribute on the Net.

Uninstall: deletes the Module. A javascript alert will show: "Are you sure you want to delete selected items?"
Click 'OK'. The page will reload displaying the word 'Success'.

NOTE: The modules are deleted from the database. They cannot be recovered from the Trash Manager. ALL the instances/copies of the Module are also totally deleted from the database. 

JS Tree

This navigation tree provides shortcuts to access various aspects of the Administration related to Modules.
Site links to the Module Manager. This is where you may edit your Site modules.
Admin links to the Module Manager. This is where you may edit your Admin modules.
Positions:
Edit Positions links to the Positions Manager.
Preview Positions opens a frame in the administrator's interface displaying your Home page template with the postions used in the template.
Install links to the Module Installer.
List packages links to a list of all Modules packages available on your site.
