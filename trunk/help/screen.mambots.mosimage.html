Mambot: MOS Image

MOS Image mambot

Usage: { mosimage }

Replaces tag in content with specified images.

You can insert as many of these tags as needed in your articles, and will then need to select the images on the "Image" tab under "MOSImage Control" in content editing or creation screens.

The replacement of the tags will follow the order of the images in the "Content Images" box.

Note: The tag described above should have no space after  "{" and before "}". It has been displayed with spaces here not to implement the function in this screen.

Toolbar Icons

Save: Click this icon to save the parameters entered and return to the Mambot Manager.

Apply: Click this icon to save the parameters entered AND keep the same window open for editing.

Close: Cancels the action. Any modification is lost.

Mambot Details

Name: Type here the name of the Mambot as it will appear on the Site Mambots listing. This has no impact on your site.

Folder: The Category into which this mambot is installed.

Mambot file: The name of the Mambot .php file as it appears in the folder. Do not change.

Order: You may change here the order of the Mambot as listed in the Folder when filtering by category.

Access Level: Choose who will be able to view this content. Public, Registered or Special.

Published: Check the box to let the Mambot show on your site or provide the function it is designed to.

Description:  Replaces { mosimage } tag in content with specified images

Parameters

Margin: Margin of Div surrounding Image & caption. Applies only when caption is used. Type a figure in px.

Padding: Padding of Div surrounding Image & caption. Applies only when caption is used. Type a figure in px.
