Web Links Manager

-- Draft 4.5.3

The Web Links Manager shows a list of the Web Links you may propose to your users through the Web Link Menu Item.

Toolbar Icons

Select Web Links by clicking the checkbox at the start of each row.

Publish and Unpublish: : Click the 'Publish' or 'Unpublish' icon to publish or unpublish any number of selected Web Links. You may check multiple checkboxes.

Delete: Lets you delete the Web Links chosen. An Alert will pop up to let you confirm this choice.

Edit : Edit an existing Web Link parameters.

New : Create a new Web Link.

Filters

From left to right:
Search a Web Link by name.
State: displays depending on Publish or Unpublished state.
Category: displays depending on the category the web link belongs to.

Column Descriptions

Clicking on any title will order the items according to the title clicked.

Check Box/Padlock: Click this to select the item. A padlock icon will replace the Check Box if the item is checked-out. Clicking on it will check-in the item on an individual basis. If this item has been checked out by another manager/administrator, please contact the administrator of your site for him/her to Check-in the item. When checked, clicking the Toolbar Icons will perform the various actions available.
 
Title: This is the name of the Web Link. Click the name to edit the Web Link.
 
Category: This is the category that the Web Link is assigned to. Click to edit the Category.
 
Published: Whether a web link is published or not. Click the icon to change the state.

Reorder : This shows the order of Web links in the list. Click the arrow icons to move the item up or down.

Order : Shows the order of Web links in the list. You may change it by editing 2 # conjointly and clicking on the ‘Save Order’ icon next to the Column Heading ‘Order’.

Hits: This is the number of hits that the Web Link has received.

ID: The web link ID in the database.
 
At the bottom of the list you may see a pagination feature, which will allow you to navigate to the next set of items in the list. You can also set the number of items displayed on a page by selecting the number in the drop down box at the bottom of the screen.
