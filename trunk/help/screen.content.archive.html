Archive Manager

draft 453

This page displays a list of all archived Content Items.

Toolbar Icons

Select items by clicking the checkbox at the start of each row.

Unarchive: returns the Item to its original location.

Trash: puts Content items in the trash bin. They are not completely removed but can't be displayed on your site. You can restore these Content Items through the Trash Manager.

JS Tree

This navigation tree provides shortcuts to access various aspects of the Administration related to Contents.
Static Content links to the Static content Manager
Content Categories are then displayed
Frontpage links to the FrontPage Manager
Trash links to the Trash Manager

Filters

From left to right:
Search an archived content by name.
Section: displays the archived contents according to the section they belong to.
Category: displays the archived contents according to the category they belong to.
Author: displays the archived contents according to their authors.

Column Descriptions

Clicking on any title will order the items according to the title clicked.

Check Box/Padlock: Click this to select the item. A padlock icon will replace the Check Box if the item is checked-out. Clicking on it will check-in the item on an individual basis. If this item has been checked out by another manager/administrator, please contact the administrator of your site for him/her to Check-in the item. When checked, clicking the Toolbar Icons will perform the various actions available.

Title: This is the title of the Archived Content Item.

Order: This shows the order of the Content Items (the "Default" "Primary Order" parameter editable through the specific Archive Menu Item). Click the arrow icons to move the item up or down in the list.

Section: This id of the Section the Item belongs to. Click to edit the Section.

Category: This id of the Category the Item belongs to. Click to edit the Category.

At the bottom of the list you may see a pagination feature, which will allow you to navigate to the next set of items in the list. You can also set the number of items displayed on a page by selecting the number in the drop down box at the bottom of the screen.
